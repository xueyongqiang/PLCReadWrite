人脸检测欢迎播报系统

基于 OpenCV + 人脸检测 + 离线语音合成的自动欢迎播报程序，检测到人脸时自动播放欢迎语音，带冷却防重复播报，可直接运行在电脑 / 树莓派上。

功能亮点

✅ 实时摄像头人脸检测
✅ 离线语音播报（无需联网）
✅ 冷却时间设置，防止重复播报
✅ 可视化人脸框显示
✅ 支持 Windows /macOS/ Linux / 树莓派

运行环境

Python 3.x
摄像头（笔记本自带 / USB 摄像头）
安装依赖
bash
运行
pip install opencv-python pyttsx3


使用说明
运行程序
bash
运行
python 你的文件名.py
程序会自动打开摄像头
检测到人脸后，会自动语音播报：欢迎光临，我已经看到你了
默认 5 秒内不会重复播报
按 q 键退出程序
自定义修改
你可以直接修改代码里的这些参数：
python
运行
# 修改语速（数值越大语速越快）
engine.setProperty('rate', 160)

# 修改冷却时间（秒）
cooldown = 5

# 修改欢迎语音
say("你想播放的语音内容")
常见问题
语音没声音
检查电脑音量
Windows 一般直接可用，Linux / 树莓派可能需要安装语音库：
bash
运行
sudo apt-get install espeak
摄像头打不开
把 cv2.VideoCapture(0) 改成 1、2 试试
检查摄像头是否被其他程序占用
退出方式
窗口选中后，按键盘 q 键即可安全退出。
